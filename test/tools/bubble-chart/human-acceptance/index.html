Welcome to Vizabi Bubble Chart (Version 0)

In order to test the API, here is a list of Test cases along with their correspondent description:

  • Test 000 - Smashed Version of Vizabi will load the application
  • Test 001 - One instance of the Vizabi will be loaded and a button will change one indicators
  • Test 002 - Two instances of Vizabi will be rendered with the same data. Just to make sure, they do not spoil one-another
  • Test 003 - Two instances of Vizabi will be rendered with the different datasets. Just to make sure, they do not spoil one-another
  • Test 004 - Description: Vizabi Test Pages: By passing the file-format option as singleCSV, Vizabi will render the whole CSV file
  • Test 005 - By passing the renderMode as print, Vizabi will render in aspect ration of an A4 paper, so that you can modify it for print purposes
  • Test 006 - By passing the file-format option as multipleCSV, Vizabi will render CSV files from multiple sources and combines them
  • Test 007 - Entities names are received by an i18n API call and will be rendered based on the language requested by user
  • Test 008 - Entities names are received by an i18n API call data will be filled by interpolation based on previous years
  • Test 009 - Vizabi can be used so that it remembers its model and attributes in the URL state
  • Test 010 - Vizabi allows users to interact with the labels in edit mode, drag labels to their desired position and remembers it in its state
  • Test 011 - In print mode, Vizabi can have a language select box which then re-draws the labels in other languages
  • Test 012 - Gapminder World 2013 tool (Full i18n Supported)
  • Test 013 - In print mode, Vizabi will change upon changing in the X-axis indicator from/to GDP from/to GNI (Full i18n Supported)
  • Test 014 - MultiCSV Support (1), First Administrative Divisions of India
  • Test 015 - MultiCSV Support (2): First Administrative Divisions of Brazil
  • Test 016 - MultiCSV Support (3): Countries & First Administrative Divisions of India and Brazil before data cooking
  • Test 017 - MultiCSV Support (4), Countries & First Administrative Divisions of India and Brazil
  • Test 018 - Fernanda's presentation at SIDA (Toward a Global Open Aid Map): Countries & First Administrative Divisions Brazil
  • Test 019 - Vizabi can show sub-national bubbles of indian and brazilian state and change the Y Indicator for them
  • Test 020 - In singleCSV file format, if there is no indicators in the model, Vizabi will pick the columns in order and sets it for the user
  • Test 021 - By setting the chart state autoZoom to true, the chart will zoom out from the selected minX and maxX value that you selected
