Welcome to Vizabi Bubble Chart (Version 0)

In order to test the API, here is a list of Test cases along with their correspondent description:

  • Test 0 - Smashed Version of Vizabi will load the application
  • Test 1 - One instance of the Vizabi will be loaded and a button will change one indicators
  • Test 2 - Two instances of Vizabi will be rendered with the same data. Just to make sure, they do not spoil one-another
  • Test 3 - Two instances of Vizabi will be rendered with the different datasets. Just to make sure, they do not spoil one-another
  • Test 4 - Vizabi can be used so that it remembers its model and attributes in the URL state
  • Test 5 - By setting the chart state autoZoom to true, the chart will zoom out from the selected minX and maxX value that you selected
  • Test 6 - By setting the chart state manualZoom to true, user can manually zoom the area of the chart which you would like to see
